# Java Console Calculator

## Objective
This project is a simple Java Console Calculator developed as part of the Java Developer Internship Task 1.

## Features
- Addition
- Subtraction
- Multiplication
- Division
- Handles division by zero
- Uses Scanner for user input
- Uses methods for each arithmetic operation
- Uses a loop to perform multiple calculations until the user exits

## Technologies Used
- Java (JDK 17 or above)
- VS Code / IntelliJ IDEA

## How to Run

1. Clone the repository:
   ```bash
   git clone https://github.com/your-username/Java-Console-Calculator.git
   ```

2. Open the project in VS Code or IntelliJ IDEA.

3. Compile the program:
   ```bash
   javac Calculator.java
   ```

4. Run the program:
   ```bash
   java Calculator
   ```
